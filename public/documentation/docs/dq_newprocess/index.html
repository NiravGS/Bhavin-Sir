DQ - Getting Started

Get Started#

Before you begin visualizing dashboards, you must connect to your data and then step up data source connection. Next step is to set up the process. Process is collection of tables / data set on which data checks can be performed, each process should be tagged with a unique name. Source name is a parent of process. A single source can be mapped to multiple process but a process can not span multiple sources.

Log-in creation#

Log-in / Create account using email id and password. Each log-in is consider owner of the process. Dashboard can be shared with other users but process don't span other accounts.

HOME#

List different sections of Wavicle data platform. Under Data quality click Launch button to create the process. To know more about data quality bot click documentation section.

LAUNCH#

Launch process allows the user to set up a new process or select the existing process to alter or view dash boards.

DATA QUALITY HOME#

CREATE CONNECTION#

Set up connection to target source system:

Host: URL for target system

User: Valid user name

Password: Valid password

Port: Valid port no based on data base type

Database: Database on which data quality to be performed

Additional settings: additional information used to establish connection. e.g. Data warehouse for Snowflake.

Test: Used to test the connection. On Successful connection a message will be displayed on top right corner.

Cancel: Return to home page.

Save: On click, connection details will be saved to DQ bot source connection for establish connection to scheduled run.
