DQ - Introduction

Data Quality#

Data quality refers to overall health of an data set on which an organization thrives to generate insights and business growth. As the size of the data grows chances of the data quality getting impacted increase along with increase in cost of monitoring. This cloud based data quality bot automates entire process of monitoring data quality. With few simple steps health of the data set can be visualized and scheduled for daily quality checks with almost no technical knowledge. This tool supports data quality checks on databases, data warehouse, aws s3 (file base).

Why data quality ?#

Broad data source connectivity: Data quality support on cloud and native databases along with cloud-based file storage systems in one single platform.

26 + data quality checks Covering all dimensions of data quality by slicing and dicing the data deep till column level.

Interactive User-friendly: User Interface to create complex data quality rules with knowledge on data as the only per-request.

Scheduled monitoring: Scheduling of data quality checks multiple times a day, weekly, monthly and being informed about the data quality as soon data is created without any manual effort.

Fully managed Data Quality Dashboard: Interactive dashboard to help decision making in matter of minutes.

Single data quality platform for everyone: deriving data about data quality at business and technical language allowing stakeholders to spend energy in making decisive strategy rather than being miss-informed.
