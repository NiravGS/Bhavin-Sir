DI - Manage Processes

Once after creating a process you will still have the ability to update and configure the processes. We provide a suite of actions that can be performed to manage and govern your processes.

Below are some of the action you can perform,

  • View/Edit/Delete Process
  • Activate/Deactivate Process
  • Run a process manually
  • Check execution history of a process

View/Edit/Delete Process#

In the Data Integration window, click the View All Process located on the left side bar. This will display all the process that has been created by your team.

As you can see above, the overview page displays the basic information about the process such as Process Name, Source Type, Source name, Target type, Target Name and Last Execution Timestamp.

You also have to ability to search for the process based on any field displayed.

Click on the process you wish to make changes. You will be shown with various options such as Edit, View, Delete etc., to the right of the screen.

View process#

Clicking on View will open up the corresponding process page with all the information related to that process including the source tables, schedule information. CDC and SCD columns if any etc.,

You still have the ability to do all the Edit, Delete, Deactivate/Activate from this view page. Look for Options at the top right end of the window.

Edit process#

Clicking on Edit will open up a wizard similar to create process where you can see all the existing values are already pre-populated for you. Edit the required fields and follow the same steps as described in the create process and save them. This will update the existing process.

You will also to Edit the process from View process page. Look for Options at the top right end of the window.

Delete process#

Clicking on Delete of corresponding process page you wish to delete, a confirmation is required by you to delete the process. Upon confirmation the process will be deleted.

You will also be able to delete the process from View process page. Look for Options at the top right end of the window.

Note: Delete is a permanent action and cannot to restored.

Activate/Deactivate Process#

All the processes when created are activated by default. In you want to pause the execution of the process for a period of time, we suggest you to deactivate the process. All the process settings including the schedule information will be retained. You can activate the process as and when needed and it will run as per the schedule. No new changes are required to restore the process.

Run a process manually#

If you ever want to run a job manually, or you get a request for an adhoc run, you can just trigger the process by clicking on the Run button either in the View process page or on the main View All process page when you click on the process. Upon execution you will see a loading next to the process indicating that the process is executing. Upon completion you will get a notification in the notification at the top right corner.

Check execution history of a process#

In the View process page, you will be able to see the Executions tab, under which you will be able to see the history of the corresponding process execution with the latest execution shown first. You will also be able to download the log file for a particular execution by clicking on the log_file_path for any further analysis.
